## 🎯 Project Overview

**VitaTrack** is a societal wellness mobile application designed to help individuals build healthier daily habits — tracking steps, hydration, sleep, and calories — while contributing to a healthier society as a whole. This repository contains the complete UI/UX design manual including high-fidelity screens and professional UX documentation.

<br/>

| 🏷️ Project | VitaTrack — Societal Wellness App |
|---|---|
| 🎨 Design Tool | Figma / Adobe XD |
| 📱 Platform | iOS 17 · Android · Mobile-first |
| 🌙 Theme | Dark Mode First — Midnight Teal |
| 🎯 Design Pattern | Card UI · Progress Rings · Bottom Nav |
| 📊 Deliverables | 5 Hi-Fi Screens + Sitemap + Journey Map + Usability Map |
| ✅ Usability Score | **8.6 / 10 — Excellent** |

|:---:|:---:|---|---|---|

## 📐 Design Patterns

| Pattern | Applied On | User Benefit |
|---|---|---|
| **Circular Progress Ring** | Home score, Activity rings, Sleep arc | Instant visual data at a glance |
| **Card-based Layout** | All content sections | Clean grouping, easy to scan |
| **Persistent Bottom Nav** | All 5 screens | Thumb-friendly, always accessible |
| **Progressive Disclosure** | Health screen (Hydration → Sleep → Tips) | Reduces cognitive overload |
| **Gamification** | Streaks, Level badges, Challenges | Drives daily engagement |
| **Sparkline Charts** | Heart rate, Weekly activity | Compact trend visualization |
| **Smart Reminders** | Hydration log, Sleep tracker | Habit formation support |
| **Adaptive Goals** | Profile goal trackers | Personalized experience |
| **Empty State Design** | No workout logged, No data yet | Guides user to take action |
| **Optimistic UI** | Log water, Mark workout done | Instant feedback, no wait |
